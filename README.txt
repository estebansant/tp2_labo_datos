Las bibliotecas reqqueridas para correr el código son:

- pandas
- numpy
- matplotlib
- scikit-learn

Ejecutar el siguiente comando para instalarlas en el proyecto:

pip install pandas numpy matplotlib scikit-learn



--------------

Lo único que debe hacerse para correr el script con éxito es asegurarse de que exista un archivo llamado "TP02-EnglishTypeAlphabet.csv" en la misma carpeta que el script del código, abrir Spyder, asegurarse de tener las dependencias instaladas y de elegir correctamente la ruta de la carpeta donde se encuentra el script, y finalmente correr todas las celdas del archivo "TP-02-UNDEFINED.py"

Si se desea ver en detalle los gráficos generados por el script, eso se encuentra dentro de la carpeta llamada "graficos".

Dentro de la carpeta "graficos", existe otra carpeta llamada "paresLetras" que contiene figuras extra que no se utilizaron en el informe, pero sí son útiles para visualizar la varianza entre dos pares de letras.

El informe del trabajo se encuentra en el archivo de "TP-02-Informe-UNDEFINED.pdf".
